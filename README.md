# Fort Carson Fire and EMS – Prehospital Practice Guidelines

This repository hosts a searchable, navigable website generated from the Fort Carson Fire and EMS **Prehospital Practice Guidelines** PDF.

## How to publish on GitHub Pages
1. Commit this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under “Build and deployment”, choose **Deploy from a branch**.
4. Select **main** and **/(root)**, then Save.
5. Open the URL GitHub provides.

## Notes
- The site includes **scanned page images** so all tables/charts/flowcharts are preserved exactly.
- Full-text search is powered by Lunr.js.
