About

Source document: Fort Carson Fire and EMS Prehospital Practice Guidelines
Revised 5/19/2022 (cover page). This website is a navigation + search layer on top of the PDF.

What you can do here

  • Search across the full text (fast).
  • Open any guideline page and read extracted text.
  • Expand “Scanned pages” on a guideline to see the original tables/charts/algorithms exactly as they appear in the PDF.

How to host on GitHub Pages

  1. Create a new repo (or use an existing one).
  2. Copy the contents of this folder into the repo root.
  3. Commit + push to GitHub.
  4. Repo Settings → Pages → “Deploy from a branch” → select main and folder / (root).
  5. Open the Pages URL GitHub gives you.
This site includes page scans (JPG) so tables and flowcharts are preserved.
